# 프로젝트 용어사전

본 프로젝트에서 **DB 컬럼명·테이블 접두사·공통 필드** 등에 사용하는 용어를 **한글명 · 영문명 · 영문약어** 표로 정리했다.

로드세이프티 **기획서·도메인 문서**와의 맞춤은 **§10-2**, **§13** 을 본다. (워크스페이스 `docs/00.프로젝트/00-04` 등과 상호 참조.)

---

## 1. 일시 · 일자

| 한글명 | 영문명 | 영문약어 |
|--------|--------|----------|
| 일시 | date time | dtm |
| 일자 | date | dt |
| 등록 일시 | register date time | reg_dtm |
| 수정 일시 | modify date time | mod_dtm |
| 생년월일 | birthday | birthday |
| 기간 시작 | from | dtm_from |
| 기간 종료 | to | dtm_to |

> `dtm_from` / `dtm_to`는 검색 조건 필드에 사용하는 접미사. 예: `rev_dtm_from`, `rev_dtm_to`.  
> Java VO에서는 camelCase로 `revDtmFrom`, `revDtmTo`.

## 2. 등록 · 수정 · 상세

| 한글명 | 영문명 | 영문약어 |

## 9. 목록 · 상세 · 등록 · 수정(URL 경로)

화면/API URL 규칙. **페이지 진입**은 `-form` 접미사, **API**는 동작명만 사용. 도메인·리소스는 용어사전 약어(adm, mber, usr 등) 사용. **등록=reg, 수정=mod, 상세=detail** 로 URL 경로 통일.

**페이지 진입(화면)**

| 한글명 | 영문명 | URL 경로 | 비고 |
|--------|--------|----------|------|
| 목록 | list | `/**/**/list-form` | 목록 화면 |
| 상세보기 | detail | `/**/**/detail-form/{id}` | 단건 조회 화면 |
| 등록 | register(reg) | `/**/**/reg-form` | 등록 폼 화면 |
| 수정 | modify(mod) | `/**/**/mod-form/{id}` | 수정 폼 화면 |

**데이터(API)** — 접두사 /api 없음. 같은 도메인/리소스 경로에서 list, detail, reg, mod 로 구분.

| 한글명 | 영문명 | URL 경로 | HTTP | 비고 |
|--------|--------|----------|------|------|
| 목록 | list | `GET /**/**/list` | GET | 목록 조회 |
| 단건 조회 | detail | `GET /**/**/detail/{id}` | GET | 상세 조회 |
| 등록 | register(reg) | `POST /**/**/reg` | POST | 등록 |
| 수정 | modify(mod) | `PUT /**/**/mod/{id}` | PUT | 수정 |
| 삭제 | delete | `DELETE /**/**/{id}` | DELETE | 삭제 |

## 13. 기획서(로드세이프티) ↔ 용어사전 통일표

기획서·도메인 MD(`docs/00.프로젝트/00-04`, `00-05`, `docs/10.DB설계` 등)에서 쓰는 말을 **본 용어사전의 한글·약어**로 모은다. 구현·DDL 작성 시 **§6 테이블 접두사**, **§3 PK `*_seq`**, **공통 필드 `reg_dtm` 등**은 `00-01.프로젝트규칙.md` §3과 함께 적용한다.

### 13-1. 역할·회원 (기획 표기 → 통일)

| 기획서·도메인 문서 표기 | 통일 한글명 | 영문 / 약어 | 비고 |
|-------------------------|-------------|-------------|------|
| 기업회원 | 차단기업(회원) | user **usr** / member **mber** | §12 `loginType=company`. VO·패키지는 `UsrMber*` 등 §2 규칙 |
| 차단기업(기업회원) | 위와 동일 | `BLOCK_CORP` 등 코드값과 병기 | |
| 차단기업 직원(부서) | 차단기업 직원 | `BLOCK_STAFF` / §12 `employee` | TMA·승인 주체가 기업과 다름(00-04) |
| 발주 담당, 발주기업 담당자 | 발주 담당자 | `order_contact` | §10-2 |

## §18. 공사 금액 모달 도메인 용어 (추가)

내 공사 현황 상세 > 진행상황 탭 > 구분/금액 클릭 시 표시되는 **공사 금액 모달(B3-1-1)**에서 도출된 신규 용어입니다.

### §18-1. 모달 필드 용어

| 한글명 | 영문명 | 영문약어 | 비고 |
|--------|--------|----------|------|
| 공사 금액 | construction amount | constr_amt | 모달 타이틀 |
| 지급액 | payment amount | pay_amt | 최종 지급 금액 (품목 합산 또는 취소 정산액) |
| 투입수량 | input quantity | input_qty | 품목별 투입 대수/명 수 |
| 시간대 | time slot | time_slot | 공사 시간대 (주간/야간) |
| 대기중 | pending | pending | 금액 미책정 상태 (운영시스템 미입력) |

### §18-2. 모달 표시 케이스

| 구분 코드 | 표시 내용 | 비고 |
|---|---|---|
| `REHEARSAL` | 구분/시간대 + 품목별 금액 테이블 + 지급액 합계 | 시연회 날짜 금액 |
| `WORK_COMPLETE` | 구분/시간대 + 품목별 금액 테이블 + 지급액 합계 | 공사 완료 날짜 금액 |
| `CANCEL` | 구분/시간대 + 지급액 단순 행 | 투입수량 단가 없이 지급액만 표시 |
| `CANCEL` (미책정) | 구분/시간대 + 지급액 `대기중` | 운영시스템 미입력 상태 |
