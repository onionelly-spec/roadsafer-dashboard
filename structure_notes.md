# 로드세이프터 대시보드 구조 분석

## 전체 레이아웃
- 좌측: 사이드바 (다크 배경 #1a1f2e 계열)
- 우측: 메인 콘텐츠 영역 (밝은 회색 배경)

## 사이드바
- 로고: "로드세이프터 SYSTEM 차단기업" (상단)
- 닫기 버튼 (< 화살표)
- 사용자 아바타: 노란 원 "나" + "나잘난" + "한국도로차단(주)"
- 메뉴 섹션: "차단기업"
  - 메인 대시보드 (홈 아이콘)
  - 내 공사 현황 (빌딩 아이콘)
  - 공사 정보 (문서 아이콘)
  - TMA 카드 관리 (카드 아이콘)
  - 장터 (쇼핑 아이콘)
  - 커뮤니티 (사람 아이콘)
  - 공지사항 (벨 아이콘)
  - 광고 (사각형 아이콘)
- 하단: 정보변경, 로그아웃

## 상단 헤더
- "메인 대시보드" 타이틀
- 우측: 노란 아바타 "나" + "나잘난"

## 메인 콘텐츠

### 1. 회사 정보 카드 (좌측)
- 로고 이미지 (로드세이프터 R 로고)
- 한국도로차단(주)
- 대표자: 나잘난
- 승인인: 김석진, 정호석, 박지민
- 차단기업 배지 (노란색)
- 공사 현황 버튼 (노란 테두리)
- 공지사항 버튼 (회색 테두리)

### 2. 퀵메뉴 (우측)
- TMA카드 관리 (선택됨 - 노란 테두리)
- 내 공사 이력
- 공사 정보
- 장터
- 커뮤니티

### 3. 오늘 알림 섹션
- 제목 + 숫자 배지 "2"
- 더보기 > 링크 (노란색)
- 알림 목록 3개:
  - [국도32번선 횡단보도 보수] 오늘 일정이 완료되었습니다. 다음 일정을 확인하세요. - 2026-06-12 11:45
  - [중앙분리대 시공] 공사 정보와 일정이 변경되었습니다. - 2026-06-11 09:00
  - [경부선 포장공사] 공사일정이 수정되었습니다. - 2026-06-10 09:00 (비활성)

### 4. 광고 이미지 섹션
- 제목 + 관리 > 링크
- 3개 이미지 (공사 현장, 전자 부품, CCTV)
- + 광고 등록 버튼 (원형)

### 5. 공사현황 테이블
- 공사현황 > 링크
- 컬럼: 상태, 공사명, 발주업체명, 기간, 씨인카 수량, 관리
- 데이터:
  - 예정 | 호남선 순천방향 포장공사 | 한국개발 | 2026-07-01~2026-07-03(3일) | TMA-1:1대, TMA-3:2대, PLCS:2대 | 차단 취소
  - 진행중 | 국도 32번선 횡단보도 보수 | 소망건설 | 2026-06-25~2026-06-25(1일) | TMA-1:2대, TMA-3:1대 | 오늘 완료
  - 진행중 | 중앙분리대 시공 | 오봉도건 | 2026-06-12~2026-06-18(7일) | TMA-3:2대, PLCS:2대 |
