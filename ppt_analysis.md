# 로드세이프티시스템 기획서 분석

## 제외 페이지 (참고만)
- 1,2,3 : 표지/목차/개요
- 15,16 : 참고/부록

## 구현해야 할 페이지 목록

### 기존 구현 완료 (유지/활용)
- login.html : 로그인 (L)
- index.html : 메인 대시보드 (A1)
- construction-list.html : 내 공사현황 목록 (B2)
- construction-detail.html : 내 공사현황 상세 (B2-1, 탭: 공사정보/차단정보/진행상황)

### 새로 구현 필요

#### B1 - 공사 정보 (차단기업 로그인 > 메인화면 > 공사 정보)
- 파일: construction-info.html
- 검색: 공사명, 발주처, 도로명 검색
- 필터: 전체/진행중/완료/대기
- 목록: 공사명, 발주업체명, 기간, 씨인카수량, 상태
- 상세보기 링크 → construction-info-detail.html

#### B1-1 - 공사 정보 상세 (차단기업 로그인 > 메인화면 > 공사 정보 > 자세히보기)
- 파일: construction-info-detail.html
- 탭: 공사 정보 / 차단 정보 / 진행 상황
- 공사정보 탭: 상태, 공사명, 발주업체명, 담당자, 기간, 시간, 도로구분, 길구분, 도로명, 방향, 공사구간, 품목수량, 사연회, 관역, 책정단가
- 차단정보 탭: 담당자, 차단구간, TMA1/TMA3/PLCS/신호수, 의견
- 진행상황 탭: 주간 캘린더 형태

#### B3 - TMA 카드 관리 (차단기업 로그인 > 메인화면 > TMA 카드 관리)
- 파일: tma-card.html
- PPT 페이지 21-23 참고
- TMA 카드 목록, 등록/수정/삭제

#### B4 - 장터 목록 (차단기업 로그인 > 메인화면 > 장터)
- 파일: market.html
- 카테고리 필터: 씨인카/간판시설물/안전시설물/씨인보드/TMA
- 검색, 내 글 보기, 신규 등록
- 그리드형 목록 (PC: 4열, 모바일: 2열)
- 페이지네이션

#### B4-1 - 장터 자세히보기
- 파일: market-detail.html
- 제목, 등록일/아이디, 수량/규격/판매가/내용, 사진
- 수정/삭제/상담신청/목록 버튼

#### B4-2 - 장터 등록/수정
- 파일: market-write.html
- 카테고리 선택, 수량/규격/판매가/내용/이미지첨부

#### B5 - 커뮤니티 목록 (차단기업 로그인 > 메인화면 > 커뮤니티)
- 파일: community.html
- 검색(선택+키워드), 내 글 보기, 신규 등록
- 번호/제목/작성자/작성일 목록
- 댓글수 표시 [N]

#### B5-1 - 커뮤니티 자세히보기
- 파일: community-detail.html
- 제목, 등록일/아이디, 내용+사진
- 수정/삭제/목록 버튼
- 댓글 기능 (대댓글 포함)

#### B5-2 - 커뮤니티 등록/수정
- 파일: community-write.html
- 제목, 내용, 이미지첨부

#### B6 - 공지사항 목록 (기업 회원 로그인 > 공지사항)
- 파일: notice.html
- 확인하지 않은 공지 / 이전 공지 구분
- 제목 클릭 시 모달 팝업 (B6-1)

#### B6-1 - 공지사항 자세히보기 (모달)
- notice.html 내 모달로 구현
- 제목, 등록일, 내용, 첨부파일, 닫기

#### B7 - 광고 목록 (로그인 > 메인화면 > 광고)
- 파일: ad.html
- 4열 그리드, 대표이미지+회사명
- 페이지네이션 없이 인피니티 스크롤

#### B7-1 - 광고 상세
- 파일: ad-detail.html
- 제목, 회사명/작성일, 광고내용(에디터), 목록

#### B8 - 정보변경 (기업회원)
- 파일: profile.html
- 공통정보: 아이디/상호/사업자번호/대표자명/연락처/이메일/주소
- 차단업체 추가정보: 권역설정/통장사본
- 저장/비밀번호변경/회원탈퇴

#### B8-1 - 정보변경 (부서/직원용)
- profile.html 내 탭 또는 별도 처리

#### B8-2 - 비밀번호 변경 (모달)
- profile.html 내 모달로 구현

## 사이드바 메뉴 구조
- 메인 대시보드 → index.html
- 내 공사 현황 → construction-list.html
- 공사 정보 → construction-info.html
- TMA 카드 관리 → tma-card.html
- 장터 → market.html
- 커뮤니티 → community.html
- 공지사항 → notice.html
- 광고 → ad.html
- (하단) 정보변경 → profile.html
- (하단) 로그아웃 → login.html
